Redkassa

Effortless ticketing for memorable events, easy buy and unforgettable experiences.

Challenge

Taking on the RedKassa project presented the captivating challenge of revolutionizing the user experience in the realm of event ticket purchases.

The task at hand involved a comprehensive revamp, introducing a cutting-edge UX search engine and minimalist UI to streamline the ticket-buying process and elevate the overall user journey.

New design concept. Now there are no extra elements. The menu was removed, thereby allowing the user to focus on searching for events.

Original brand colors

The font is more readable

Approach

My strategy involved an in-depth exploration of user behavior within the context of event ticketing. By immersing myself in user preferences and studying industry best practices, I meticulously shaped the project's core features. Purposeful removal of superfluous elements, such as the menu, aimed to guide user attention to the primary objective: effortlessly finding and securing tickets for desired events.

Visuals and Design Principles

The transformation of RedKassa interface centered on a strategic emphasis on simplicity and clarity. The UX search engine, characterized by a minimalist UI, ensures a seamless navigation experience. The addition of a like system empowers users to curate a personalized list of favored events for future reference and purchases.

Now in Redkassa you can create your own account! Designed forms for registration and authorization

A system of likes has been introduced for events interesting to the user in order to save them and return to them later for future purchases

Results

In conclusion, the RedKassa project stands as a successful endeavor in redefining the ticket-buying experience. The implementation of a minimalist UI, and personalized features has transformed the platform into an intuitive space for users to effortlessly discover, like, and secure tickets for their favorite events.

This redesign not only enhances user satisfaction but also positions RedKassa as a modern and user-friendly solution in the event ticketing landscape.
